# Rah-E-Haq Website

A premium, modern NGO/charity homepage built with React, Tailwind CSS, and Framer Motion.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- **React 18** - UI framework
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Vite** - Build tool
- **React Icons** - Icon library

## Features

- ✅ Fully responsive design
- ✅ Scroll-triggered animations
- ✅ Glassmorphism effects
- ✅ Counter animations
- ✅ Testimonial carousel
- ✅ Masonry gallery with lightbox
- ✅ Newsletter subscription
- ✅ Trust metrics dashboard
- ✅ Mobile-first approach
- ✅ SEO optimized

## Project Structure

```
├── src/
│   ├── components/     # React components
│   ├── App.jsx          # Main app
│   ├── main.jsx         # Entry point
│   └── index.css        # Global styles
├── public/              # Static assets
├── index.html           # HTML template
├── tailwind.config.js   # Tailwind config
└── package.json
```

## Customization

### Colors
Edit `tailwind.config.js` to customize the color palette.

### Content
Update text in respective component files:
- `Hero.jsx` - Main headline and CTAs
- `Impact.jsx` - Program cards
- `Mission.jsx` - Mission statement
- `Stories.jsx` - Testimonials
- `Trust.jsx` - Metrics and activities
- `Footer.jsx` - Footer content

## Deployment

### Vercel (Recommended)
1. Go to [vercel.com](https://vercel.com)
2. Import project or drag `dist` folder
3. Deploy!

### Netlify
1. Go to [netlify.com/drop](https://netlify.com/drop)
2. Drag `dist` folder
3. Done!
